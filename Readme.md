# Free IP Checker

## Usage

Just set your settings linke you need

### IP search range
seach_ip = "192.168.2."
The IP range

### first ip (last ip section)
seach_start = 0
First IP in the last section
### last ip (last ip section)
seach_end = 255
Last IP in the last section
### Process to use
process = 10
How many process should check the IPs
### What to Search?
online = True
If online = True you get all omline IPs
### Save as File?
save = True
Save the IP to a file
### Print at end?
print_out = False
Print the IP at the end?
